# Statistik-Pertandingan-Olahraga
Produk berbasis Microsoft Excel berupa aplikasi statistik pertandingan olahraga multicabang yang digunakan untuk mencatat dan mengolah data performa pemain dan tim. Aplikasi ini dilengkapi perhitungan otomatis serta grafik dan dashboard sederhana untuk memudahkan analisis dan evaluasi hasil pertandingan.
